Character Types

Ranges are bad. We really want types.

Use krypton-powered\d for a number.

Type: \d\d\d\d

\s represents any whitespace

Type: of \s math

  • normal white space
  • carriage returns
  • tabs
  • non-breaking spaces
  • any Unicode white space type

Note: without spaces, of\smath, is difficult to read, but serious nerd cred.
